### 第1周：基础模块实现

#### 第1-2天：项目初始化 + 磁盘I/O层

**目标**：实现页级别的磁盘读写能力

```c
// disk_manager.c 核心功能
```

**设计要点**：
- 固定页大小（如4KB）
- 使用位图(bitmap)管理空闲页
- 文件头存储元信息（页数量、空闲页位图等）

#### 第3-4天：B+树索引实现

**目标**：实现内存中的B+树，支持基本操作

```c
// bplus_tree.c 核心功能
// 1. 插入键值对
bpt_insert(BPlusTree* tree, int key, record_id_t rid);
// 2. 查找键值对
record_id_t bpt_search(BPlusTree* tree, int key);
// 3. 范围查询
bpt_range_query(BPlusTree* tree, int low, int high, Callback cb);
// 4. 删除键值对
bpt_delete(BPlusTree* tree, int key);
```

**设计要点**：
- **阶数选择**：设为d=5（每个节点最多10个键）
- **内部节点**：只存键和子节点指针
- **叶子节点**：存键和记录ID，双向链表连接
- **分裂条件**：节点键数 >= 2d
- **合并条件**：节点键数 < d

#### 第5天：缓冲池实现

**目标**：实现基于LRU的页缓存

```c
// buffer_pool.c 核心功能
// 1. 获取页（如果不在缓存则从磁盘加载）
Page* bp_get_page(page_id_t page_id);
// 2. 标记页为脏页
bp_mark_dirty(page_id_t page_id);
// 3. 刷新脏页到磁盘
bp_flush_page(page_id_t page_id);
// 4. 关闭缓冲池（刷新所有脏页）
bp_close();
```

**设计要点**：
- 采用**LRU链表**管理缓存淘汰
- 使用**哈希表**加速页查找
- 页帧结构：`data[PAGE_SIZE] + page_id + is_dirty + pin_count`

#### 第10天：查询执行器

**目标**：执行AST，调用存储引擎完成操作


**实现策略**：
1. **INSERT**：写入记录到数据页，同时更新B+树索引
2. **SELECT**：根据WHERE条件决定全表扫描或索引查找
3. **DELETE**：标记记录为删除（懒删除），更新索引
4. **UPDATE**：删除旧索引项，写入新记录，插入新索引项

## 四、关键技术难点与解决建议

### 🔴 B+树节点分裂

**难点**：分裂时需要更新父节点，可能触发级联分裂

**解决方案**：
```c
// 递归分裂策略
Node* bpt_split(Node* node) {
    // 1. 创建新节点
    Node* new_node = create_node();
    // 2. 将后半部分键移动到新节点
    int mid = node->key_count / 2;
    memcpy(new_node->keys, node->keys + mid,
           sizeof(int) * (node->key_count - mid));
    // 3. 如果是内部节点，移动子指针
    if (!node->is_leaf) {
        memcpy(new_node->children, node->children + mid + 1,
               sizeof(Node*) * (node->key_count - mid));
    }
    // 4. 更新父节点
    return new_node;
}
```

### 🟠 LRU缓冲池实现

**难点**：高效实现LRU淘汰策略

**解决方案**：
```c
// 双向链表 + 哈希表
typedef struct LRUNode {
    Page* page;
    struct LRUNode* prev;
    struct LRUNode* next;
} LRUNode;

// 访问页时，将其移动到链表头部
void lru_access(LRUNode* node) {
    // 从当前位置移除
    node->prev->next = node->next;
    node->next->prev = node->prev;
    // 插入到头部
    node->next = head->next;
    head->next->prev = node;
    head->next = node;
    node->prev = head;
}
```
